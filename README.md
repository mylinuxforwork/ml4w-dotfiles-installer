# ML4W Dotfiles Installer

Bash script to install dotfiles configurations on Arch, Fedora and openSuse.

## Installation

make install

### Dependencies

Install the development tools for your system

# Arch
sudo pacman -S --needed base-devel

# Fedora
sudo dnf groupinstall "Development Tools"

# openSuse Tumbleweed
sudo zypper install -t pattern devel_basis
